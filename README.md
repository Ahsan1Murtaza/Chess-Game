♟️ Java Chess Game – Built from Scratch

Overview
This is a fully functional Chess game implemented from scratch in Java, featuring all standard chess rules and mechanics. The game supports valid moves, check/checkmate detection, castling, en passant, pawn promotion, and more.

Features
✅ Valid piece movements & rule enforcement
✅ Check & Checkmate detection
✅ Stalemate handling
✅ En passant
✅ Pawn Promotion
✅ Castling
✅ Move Timer for both players
