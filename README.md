# Java Chess Game

A **chess game built from scratch in Java**, featuring standard chess rules, drag-and-drop piece movement, timers, and turn indicators. Fully implemented using Java Swing with no external libraries.

---

## Features

* 8×8 chessboard with proper initial setup
* Drag-and-drop movement for all pieces
* Special moves: Castling, En Passant, Pawn Promotion
* Detects Check, Checkmate, and Stalemate
* Turn indicator for White / Black
* Timers for both players

---

## Installation & Run

1. Clone the repository:

```bash
git clone https://github.com/Ahsan1Murtaza/Chess-Game.git
```

2. Open in IntelliJ or Eclipse (or compile via command line).
3. Run `Main.java` to start the game.
